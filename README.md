# Oomfiecity Tap

## How do I install these formulae?

`brew install oomfiecity/tap/<formula>`

Or `brew tap oomfiecity/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "oomfiecity/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
